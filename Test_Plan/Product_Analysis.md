Product Overview

Product Name: OpenCart Demo
Product Type: E-commerce web application

Goal:
The application allows users to browse products, add items to the shopping cart and complete the purchase process.

Target Users:

Guest users

Registered users

Platform:

Web application

Desktop browsers

Scope
In Scope:

UI elements verification

Functional testing of core user flows

Input validation

Error handling and system messages

Guest and registered user scenarios

Desktop browser behavior

Out of Scope:

Payment processing (real payments)

Backend testing

Database testing

Security testing

Admin panel functionality

Risks & Assumptions
Assumptions:

The demo website is stable and accessible

Test data can be reused

No access to backend or database is required

Risks:

Demo environment may be unstable

Real payment functionality is not available

Possible periodic data reset on the demo site
